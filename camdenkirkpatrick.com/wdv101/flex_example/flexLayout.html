What is C++?

C++ is a high-level programming language that was developed as an extension of the C programming language. It was created by Bjarne Stroustrup in 1979 at Bell Labs as an enhancement to the C language. C++ is known for its efficiency, flexibility, and performance, making it suitable for a wide range of applications, including systems software, game development, desktop applications, mobile apps, and more.

Key features of C++ include:
  1. Object-Oriented Programming (OOP): C++ supports classes and objects
  2. Memory Management: Unlike many high-level languages, C++ provides control over memory management.
  3. Performance: C++ is known for its high performance, since it doesn't need a virtual machine or interpreter.
  4. Portability: C++ code can be compiled and executed on various platforms.
  5. Standard Template Library (STL): C++ includes a powerful library called the Standard Template Library (STL), which gives developers access to various data structures.

Setting up your first program

To create a C++ program, open your IDE of choice, and create a file ending in ".cpp" (for example: helloWorld.cpp). Once you have done that, you will need to make sure that you have a compiler that is compatible with C++ installed, such as G++. After you have installed and setup the comiler, you can begin writing your first program in C++. In order for your program to input and output, you must write "#include <iostream>" at the beginning of your program. The "#" symbol tells your program that whatever follows it is part of the Standard Template Library (STL), which is where useful C++ features are stored. After that, you will write the "main function", where the program will be executed. To write the main function, type: "int main() {return 0;}". This tells the program that if there are no errors in your code, then your program should return "0" from the main function, which will end your program. This is the setup for every C++ program. So far, your C++ program should look like this: 

#include <iostream>

int main() {

  return 0;
}

Variables, Output, and Execution

There are five basic data types that are important to know in C++:

  • Integers(int): stores an integer value Ex.(-1, 0, 1)
  • Float(float): stores a number with a fractional part Ex.(1.25, 0.67, -5.2)
  • String(string): stores an array of characters Ex.("Hello", "Bye!", "100")
  • Character(char): stores a single character Ex.('A', '&', '4')
  • Boolean(bool): stores the value "True" or "False"
To declare a variable, you must write the type ( See text in parentheses above), name the variable, and assign it a value, followed by a ';'. For example: int x = 20; If you are declaring a string, you must put the value in double quotes, and if you are declaring a char, you must put the value in single qoutes (See above) In order to ouput to the console, do: std::cout << "text you want to print;" If you want to print out one of your variables, just do: "std::cout << varName;" The last step is to compile and execute your program! To compile your program using G++, type the following into your terminal: "g++ fileName.cpp -o executableName" The "executableName" will be the name that the executable file is assigned to (default is "a.exe"). Now that you have compiled the program, you can now execute it. To run your program, type the following into your terminal: "executableName"
